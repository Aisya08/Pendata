# Bussiness-Understanding

## **Latar Belakang**

Dataset ini berisi data karakteristik bunga iris yang terdiri dari tiga spesies, yaitu *Iris setosa*, *Iris versicolor*, dan *Iris virginica*.

Setiap spesies diukur berdasarkan empat atribut numerik:

* Sepal length (panjang sepal)
* Sepal width (lebar sepal)
* Petal length (panjang petal)
* Petal width (lebar petal)

Permasalahan dalam analisis ini adalah bagaimana memanfaatkan data morfologi tersebut untuk mengklasifikasikan spesies bunga iris secara akurat. Hal ini penting karena beberapa spesies memiliki kemiripan bentuk secara visual, sehingga diperlukan pendekatan berbasis data untuk meningkatkan ketepatan identifikasi.

---

## **Tujuan Bisnis**

Tujuan dari analisis ini adalah:

1. Mengidentifikasi pola hubungan antara ukuran sepal dan petal dengan spesies bunga iris.
2. Membangun model klasifikasi yang mampu memprediksi spesies iris berdasarkan atribut morfologi.
3. Mendukung proses identifikasi spesies secara lebih cepat, objektif, dan konsisten dibandingkan pengamatan manual.

---

## **Manfaat**

1. Memberikan pemahaman tentang penerapan metodologi CRISP-DM dalam kasus klasifikasi data.
2. Menjadi contoh implementasi machine learning untuk pengenalan pola.
3. Mendukung pengembangan sistem klasifikasi otomatis berbasis data.
